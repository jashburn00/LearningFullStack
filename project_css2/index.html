How to Use Shadows in HTML

Shadows can contribute to interesting and visually appealing styles, unlike this font. Here is a cheat sheet for how to use them.

Shadows use five values separated by spaces. The third and fourth are optional, and default to 0. There is also an optional prefix called "inset", which makes the shadow effect appear inside the element. Therefore, you can declare a shadow attribute with 5, 3, 6, or 4 values.
