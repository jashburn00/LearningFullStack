Colors

The colors in this style guide are inspired by space exploration, in case you couldn't tell. With the theme of "Lasers", the most prominent color for visual effects should be red, white for readability, and black for backgrounds to make the other colors stand out. The gray color, while included, is used mainly for linear gradients to give life to the headers.

Fonts

The font family used is:

Abnes,

Verdana,

and plain old sans serif.

The headers use a font called "Abnes", and its creator is kind enough to let us use it without a license as long as it is not used for profit. The Abnes font screams "LASERS" (and space exploration), and is perfect for this style. However, since it does not have great readability, it is used solely for headers. Another thing to note about the use of Abnes in this style guide is that we use a linear gradient CSS trick to add some life to the letters, rather than declaring a text color.

Verdana is a much more readable font, and is used for paragraphs. Verdana is not only a web safe font, but it is among the most legible. Thanks to its plain, clean type void of serif, it is very to-the-point and fits well with a theme of space exploration.

Finally, we rely on plain sans-serif if all else fails; plain sans-serif font is very similar to Verdana and provides much of the same appeal.

Navbar and Buttons

The navbar is stuck to the top of the screen for accessibility. The buttons contained in it are largely similar to how buttons would be used elsewhere on a page, and use similar design to the rest of the site. Their style comes almost entirely from the glow effect. When dormant, the buttons display only their text, but when hovered will display and indicative glow. The buttons gain an additional level of glow when active (being clicked on) as their form feedback to user interaction.

I am a navbtn

Buttons on the rest of the page, however, may be inclined to use dormant shadow effect or some other indicator that it is a button. The "btn" class (not to be confused with "button") uses an additional layer of shadow to indicate hover and one more for active.

I am a btn

Media Containers

The media containers used in this style guide do not inherently use a shadow effect. Rather, they are intended to use whichever glow class they want for each instance.

There are also captions to be used with the media containers, under the class "caption". The captions are intended to hold several smaller elements and by default are set to be in the bottom-right corner of their container.

The media containers are meant to be styled individually, rather than making them more suited to any one type of media.
